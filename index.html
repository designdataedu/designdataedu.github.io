Grasp the power of data science.

Design Data democratizes the learning of foundational steps in data science. It focuses on allowing learners to make sense of data through a contextualized learning environment.

Learn More

Start your data science journey

Often characterized by advanced coding skills or unrelatable datasets, learning about data science can be inaccessible for non-technical learners. Design Data changes that.

How it works

By the end of the end of this experience, you will be able to:

01
Understand the perspective that data is socially constructed
02
Design and frame a question to guide data exploration
03
Increase confidence in ability to identify patterns and extract insights based on interaction with data
04
Communicate insights from data you have interacted with
